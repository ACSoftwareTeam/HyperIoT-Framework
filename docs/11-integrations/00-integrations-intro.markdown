# Integrations [](id=integrations)

HyperIoT Framework supporto different types of integrations:

* WebSockets
* DataStreaming
  * Apache Kafka
  * Apache Storm
* Big Data
  * Apache Hadoop
  * Apache Spark
* Consensus And Coordination 
  * Apache Zookeeper Connector
* No SQL
  * HBase Connector
* Block Chain
  * Ethereum Connector
